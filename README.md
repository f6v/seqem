## SeqEM Docker

SeqEM is a genotype-calling algorithm for NGS data. This project allows running `seqem` binary locally without having to compile it from source.

Please cite the original publication:

> E. R. Martin, D. D. Kinnamon, M. A. Schmidt, E. H. Powell, S. Zuchner, R. W. Morris, SeqEM: an adaptive genotype-calling approach for next-generation sequencing studies, Bioinformatics, Volume 26, Issue 22, 15 November 2010, Pages 2803â€“2810, https://doi.org/10.1093/bioinformatics/btq526
